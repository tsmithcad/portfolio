Excel Workbook (VBA)

LED Lighting System Quote Tool

Goal: Remove the need for engineers, CAD drafters and estimators to quote simple LED Reveals and Intersections.

Solution: Create an Excel spreadsheet quote tool to assist CSR's with selecting options for a product to meet the individual needs of their client. Instead of creating quotes and specifications manually, the quote tool automates the process, speeding up the quote timeline and getting the item into production.

Microsoft Excel VBA Microsoft Excel VBA (Visual Basic for Applications) is a programming language integrated within Excel, allowing users to automate tasks, create macros, and enhance functionality. With VBA, users can write custom code to manipulate data, automate repetitive processes, create interactive forms, and extend Excel's capabilities. VBA provides access to Excel's object model, allowing users to interact with worksheets, cells, charts, and more. It's a powerful tool for customization, process automation, and building user-friendly applications within Excel.
